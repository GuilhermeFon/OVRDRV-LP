# Hero Videos

Coloque aqui os arquivos do vídeo da hero section:

- `hero-garage.mp4` — codec H.265 (HEVC), container MP4
- `hero-garage.webm` — codec VP9, container WebM (fallback)

## Especificações recomendadas

- Resolução: máx 1920×1080 (1080p)
- Bitrate: 3–5 Mbps
- Duração: 6–8 segundos
- Sem áudio (vai ser mutado de qualquer forma)
- Frame rate: 30 fps

## Conceito visual

- Frame inicial: garagem industrial fechada, portão de metal vertical, neons em kanji magenta+ciano
- Meio: portão sobe revelando o R34 azul/roxo candy paint
- Frame final: carro com xênon aceso, neon underglow pulsando, fumaça atmosférica
